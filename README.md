# SlashData Assessment

THis is my submission for the SlashData Assessment. I used the full 4-hours that was recommended and had a great time doing it!

## Installation

Clone the git repository:

```bash
# git clone https://github.com/petesfrench/SlashData-Assesment.git
```

Navigate to the root folder and install dependencies:

```bash
# npm i
```

Navigate to server folder and start

```bash
# npm run start
```

Navigate to client folder and start

```bash
# npm start
```

Client will now be runnig locally at http://localhost:3000

## Usage

You must input at very least a string into the field labelled as body, the rest can be left blank and will be filled with defults.
The defaults are as follows:
_ngram:_ 1
_case_sensitive_: yes
_length_: 100
